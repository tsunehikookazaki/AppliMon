# AppliMon10

電気機器（洗濯機、扇風機など）の動作状態を監視し、通知するAndroidアプリです。
SwitchBotプラグ等の消費電力を監視して、動作開始や終了を音声と通知でお知らせします。

## 主な機能
- 消費電力に基づいた動作状態の監視
- テキスト読み上げ（TTS）による音声通知
- フォアグラウンドサービスによるバックグラウンド監視
- 機器ごとの名前設定
- 暗号化された設定保存（トークン、シークレット等）

## セットアップ
1. SwitchBotアプリからトークンとクライアントシークレットを取得します。
2. 対象デバイスのデバイスID（MACアドレス）を確認します。
3. 本アプリの「外部連携設定」に上記を入力して保存します。
